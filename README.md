# Modern React Portfolio

A sleek, state-of-the-art personal portfolio built with React.js, Tailwind CSS, and Lucide Icons. Designed for high performance, smooth interactions, and a premium professional aesthetic to showcase full-stack development and SEO content writing skills.

## 🚀 Features
- **Single Page Application (SPA):** Built seamlessly utilizing React hooks.
- **Modern UI/UX:** Responsive layouts engineered with Tailwind CSS, utilizing glassmorphism, soft gradients, and robust drop shadows.
- **Interactive Project Showcase:** Image arrays rendered in a dynamic full-screen image modal slider with dot navigations.
- **Article Viewer Modal:** Read extensive written content seamlessly without navigating away from the main homepage.
- **Performance Optimized:** Uses Vite for lightning-fast HMR and minimal production build sizes.

## 🛠 Tech Stack
- **Framework:** React.js (Vite)
- **Styling:** Tailwind CSS (with arbitrary value support and custom PostCSS configs)
- **Icons:** Lucide React
- **Typography:** Plus Jakarta Sans (Google Fonts)

## 📦 Local Installation & Setup

1. Clone the repository
   ```bash
   git clone https://github.com/codedev-mi/sample-portfolio.git
   ```
2. Navigate to the project directory
   ```bash
   cd sample-portfolio
   ```
3. Install dependencies
   ```bash
   npm install
   ```
4. Start the Vite development server
   ```bash
   npm run dev
   ```
5. Open up `http://localhost:5173/` in your browser.

## 🏗 Build for Production
To bundle the application for production, run:
```bash
npm run build
```
This will compile the optimized application logic inside the `dist/` root folder, which you can then deploy to Vercel, Netlify, or any static web host.
